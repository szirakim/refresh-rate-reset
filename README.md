**Introduction**

Open source helper application for resetting refresh rate. I wrote it due to my S21 Ultra screen problems, when phone stuck at low or high value constantly, just run the app and press Reset.
My applications contain minimal permissions, no tracking, no ads etc.

**Permissions**

```
<uses-permission android:name="android.permission.WRITE_SETTINGS" />
```
